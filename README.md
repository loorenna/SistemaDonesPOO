# Sistema de Entregas con Drones Autónomos

Proyecto desarrollado en Java aplicando Programación Orientada a Objetos.

## Conceptos aplicados

- Herencia
- Encapsulamiento
- Polimorfismo
- Métodos abstractos
- Sobrescritura (@Override)
- Getters y Setters
- Validaciones
- Uso de Scanner

## Estructura del proyecto

- Dron.java → clase abstracta
- DronLiviano.java
- DronCarga.java
- DronEmergencia.java
- Main.java

## Funcionalidades

- Registro de datos por teclado
- Validación de información
- Cálculo de costos de entrega
- Demostración de polimorfismo
- Uso de valores quemados
- Actualización de datos mediante setters

## Cómo ejecutar

1. Abrir el proyecto en IntelliJ IDEA
2. Ejecutar la clase Main.java
3. Ingresar los datos solicitados en consola
